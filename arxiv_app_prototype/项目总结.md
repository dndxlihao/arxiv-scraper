# arXiv文献推荐APP - 项目总结

## 🎯 项目概述

基于小红书/知乎风格的arXiv文献推荐移动应用，将学术文献阅读与社交协作相结合，为研究人员提供智能化的文献发现和管理工具。

## ✨ 核心亮点

### 1. 小红书式用户体验
- **Feed流设计**：类似小红书的卡片式展示
- **智能推荐**：基于用户兴趣的个性化推荐
- **关键词标签**：自动提取和展示论文关键词
- **收藏系统**：个人文献库管理

### 2. 团队协作功能
- **团队空间**：创建和管理研究团队
- **文章推荐**：老板/同事推荐文章
- **待阅读标签**：协作阅读管理
- **任务提醒**：阅读任务和截止日期

### 3. 智能服务
- **每日简报**：早上9点推送10篇优质论文
- **关联推荐**：每篇文章推荐10篇相关文献
- **定时爬取**：自动获取最新arXiv论文
- **质量筛选**：基于多维度评分算法

## 🏗️ 技术架构

### 后端技术栈
```
FastAPI (Python Web框架)
├── SQLite/PostgreSQL (数据库)
├── Redis (缓存 + 消息队列)
├── Celery (异步任务)
├── arXiv API (数据源)
└── 自定义爬虫 (数据获取)
```

### 前端技术栈
```
React Native (跨平台移动端)
├── React Navigation (路由)
├── React Native Vector Icons (图标)
├── Redux/MobX (状态管理)
└── 自定义UI组件
```

### 部署架构
```
Docker容器化
├── 后端服务 (FastAPI)
├── Redis服务
├── Celery Worker
├── Celery Beat
└── 前端开发服务器
```

## 📱 功能模块详解

### 1. 用户系统
- **注册登录**：邮箱/用户名注册
- **兴趣管理**：设置研究兴趣关键词
- **个人中心**：收藏统计、阅读历史

### 2. 内容系统
- **论文爬取**：定时获取arXiv最新论文
- **智能推荐**：基于TF-IDF + BERT的推荐算法
- **详情展示**：完整论文信息 + 图表展示
- **关联文献**：基于主题相似度的相关推荐

### 3. 社交系统
- **团队协作**：创建团队、邀请成员
- **文章推荐**：向团队推荐文章
- **评论讨论**：论文讨论区
- **阅读任务**：协作阅读管理

### 4. 通知系统
- **每日简报**：早上9点定时推送
- **任务提醒**：阅读任务截止提醒
- **团队通知**：新推荐文章通知
