## 🔧 技术实现细节

### iOS应用架构
```swift
// MVVM架构模式
View (SwiftUI) ←→ ViewModel ←→ Service ←→ API
```

### 关键技术点
1. **状态管理**：使用@StateObject和@EnvironmentObject
2. **网络请求**：Alamofire + async/await
3. **安全存储**：KeychainAccess
4. **错误处理**：自定义错误类型 + 用户友好提示

### 后端架构
- **框架**：FastAPI
- **数据库**：PostgreSQL + SQLAlchemy
- **认证**：JWT Token
- **API设计**：RESTful风格
